* Violas wallet RESTful API Web service
** API调用路径
   前缀： =https://xxx.xxx.xxx.xxx:xxxx=

   完整调用路径为： =前缀= + =具体接口路径=
** 接口调用说明
   - 如果参数格式是 =JSON= ：提交request请求时必须添加header头： =Content-Type:application/json=
   - 请求头参数 =location= 国际化参数说明：在请求头中加入 =location= 字段用于判断返回错误信息语言
     - 英文： EN
     - 中文： CN
     - 繁体中文: CNT
** 接口说明
